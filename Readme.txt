Описание проекта:
Интернет-магазин продаёт товары нескольких категорий.
Отчёт за прошлый период показал, что активность покупателей начала снижаться.
Привлекать новых клиентов не эффективно: о магазине знает большая часть целевой аудитории.
Возможный выход — удерживать активность постоянных клиентов, возможно с помощью персонализированных предложений.
Руководство требует предложений только на основе анализа данных и бизнес-моделирования.

Цели исследования:
Разработать решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность.

Задачи исследования:
1. промаркировать уроень (снижения) финансовой активности постоянных покупателей в зависимости от количества приобритаемых товаров;
2. выделить данные о клиентах по следующим группам метрик: Коммуникации, Продуктового поведения, Поведения на сайте, Финансового поведения;
3. подбор модели машинного обучения для предсказания вероятности снижения покупательской активности клиента в последующие три месяца;
4. дополнить данными о прибыльности клиентов;
5. С учётом результатов моделирования и данных о прибыльности, выделить сегменты покупателей и разработать для них персонализированные предложения.

Используемые модели МО:
1. Метод k-ближайших соседей;
2. Метод классификации Дерева решений;
3. Метод опорных векторов;
4. Логистическая регрессия (линейная классификация).

Результат применения моделей МО:
Лучшей моделью оказалась Метод опорных векторов SVC. 
Значение метрики roc-auc на тестовой вборке составило 91.6%
Сразу за лучшей моделью с небольшим отрывом следуют методы k-ближайших соседей.
